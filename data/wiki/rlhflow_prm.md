# RLHFlow-Llama3.1-8B PRM

**Type:** model

An existing PRM based on Llama 3.1 8B trained on GSM8K and MATH data with error labels acquired via Monte Carlo roll-outs using DeepSeek or Mistral data.

---

RLHFlow-Llama3.1-8B PRM is an existing process reward model based on the Llama 3.1 8B architecture, trained on mathematical reasoning data from GSM8K and MATH datasets with step-level error labels acquired via Monte Carlo roll-outs using solution data generated by DeepSeek or Mistral models. It represents a strong baseline for PRM evaluation, having been trained on a large volume of math-domain supervision. In the FoVer paper, RLHFlow-Llama3.1-8B PRM is used as the primary comparison point for the Llama-based FoVer-PRM across the 12-benchmark evaluation suite. Comparisons between FoVer-PRM and RLHFlow demonstrate that formally verified training data can match or exceed the quality of Monte Carlo-derived labels for PRM training.
