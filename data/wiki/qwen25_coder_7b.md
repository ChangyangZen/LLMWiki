# Qwen2.5-Coder-7B

**Type:** model

A compact 7B-parameter base language model with strong code semantic understanding and mathematical reasoning capability, used as the foundation for fine-tuning AutoReal-Prover.

---

Qwen2.5-Coder-7B is a compact 7-billion-parameter base language model developed by Alibaba, designed with strong capabilities in code understanding, generation, and mathematical reasoning. It serves as the foundation model from which AutoReal-Prover is fine-tuned, providing a balance between model capacity and deployment efficiency. Its pre-training on large code corpora gives it familiarity with formal syntax and structured logical expressions, making it a suitable starting point for specialization in Isabelle/HOL theorem proving. The choice of a 7B-parameter base model enables local deployment of AutoReal-Prover without reliance on expensive cloud-based API services.
