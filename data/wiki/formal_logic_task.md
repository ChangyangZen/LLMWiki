# Formal Logic Task

**Type:** concept

A reasoning task defined using formal syntax and rules, such as first-order logical entailment, where solutions can be verified using automated solvers.

---

A formal logic task is a structured reasoning problem defined using formal syntax and inference rules, such as tasks involving first-order logical entailment, in which a solver must derive conclusions from a set of premises through a sequence of valid deductive steps. The formal nature of these tasks enables automated tools, such as the Z3 SMT solver, to verify the correctness of each reasoning step with mathematical certainty, making them well-suited for generating reliable PRM training labels. In the FoVer framework, formal logic tasks are sourced from the FLDx2 dataset and used as one of the two primary task types for constructing FoVer-40K. Solutions to these tasks are generated by LLMs and subsequently annotated step by step using Z3.
