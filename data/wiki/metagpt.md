# MetaGPT

**Type:** model

A multi-agent framework integrating human workflow patterns and standardized operating procedures into agent collaboration.

---

MetaGPT is a multi-agent framework that integrates human workflow patterns and standardized operating procedures (SOPs) into agent collaboration, drawing inspiration from structured organizational processes in software engineering and other professional domains. By encoding domain-specific workflows as agent roles and interaction protocols, MetaGPT enables teams of agents to tackle complex tasks such as software development through structured division of labor. The framework is compared to Agent Contracts in the context of multi-agent governance, with MetaGPT emphasizing workflow structure and role specialization while Agent Contracts focus on formal resource governance and conservation properties. Together, these approaches represent complementary dimensions of multi-agent system design.
