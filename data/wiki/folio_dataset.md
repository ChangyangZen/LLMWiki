# FOLIO Dataset

**Type:** dataset

A natural language reasoning benchmark with first-order logic annotations used to evaluate logical reasoning capabilities.

---

FOLIO is a natural language inference benchmark in which each example consists of a set of premises and a hypothesis expressed in ordinary English, accompanied by expert-authored first-order logic annotations that make the underlying logical structure explicit. The dataset is designed to test whether models can perform multi-step deductive reasoning grounded in formal logic, going beyond the surface-level linguistic patterns that simpler NLI benchmarks permit. FOLIO is one of six benchmarks used to evaluate VERGE, selected because its logical structure aligns naturally with the formal verification capabilities of the SMT-based pipeline. Performance on FOLIO provides insight into a system's ability to bridge natural language understanding and rigorous logical inference.
