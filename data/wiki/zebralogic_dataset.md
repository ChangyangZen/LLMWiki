# ZebraLogic Dataset

**Type:** dataset

A benchmark testing LLM logical reasoning through constraint satisfaction problems involving spatial relations.

---

ZebraLogic is a benchmark that evaluates logical reasoning through constraint satisfaction problems inspired by the classic Zebra Puzzle, in which models must determine the unique assignment of attributes to entities satisfying a set of spatial and relational constraints expressed in natural language. The problems require systematic case analysis and constraint propagation, skills that formal solvers handle naturally but that challenge purely neural approaches. ZebraLogic is included in the VERGE benchmark suite as a representative of combinatorial constraint reasoning, where the SMT-based verification pipeline can provide strong formal guidance. Performance on ZebraLogic reflects a system's ability to translate relational natural language constraints into executable logical representations.
