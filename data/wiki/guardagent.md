# GuardAgent

**Type:** method

A safeguard framework for LLM agents using knowledge-enabled reasoning and policy filters to enforce safety constraints.

---

GuardAgent is a safeguard framework for LLM agents that employs knowledge-enabled reasoning and policy filters to enforce safety constraints on agent behavior. The framework uses an LLM-based reasoning module to evaluate proposed agent actions against a set of safety policies, blocking or modifying actions that are deemed unsafe according to the specified constraints. GuardAgent represents a knowledge-driven, reasoning-based approach to LLM agent safety, contrasting with the probabilistic model-based approach of ProbGuard. It is cited as a related work in the ProbGuard paper, situating ProbGuard within the broader landscape of LLM agent safety research.
