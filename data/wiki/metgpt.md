# MetaGPT

**Type:** model

A multi-agent framework using Standard Operating Procedures as essentially compiled workflows, reducing cascading hallucinations by constraining agent behavior with deterministic procedures.

---

MetaGPT is a multi-agent LLM framework that structures agent collaboration around Standard Operating Procedures (SOPs), encoding organizational workflows as explicit procedural constraints that govern agent behavior and communication. By embedding SOPs into the multi-agent architecture, MetaGPT reduces the cascading hallucination problem common in unconstrained multi-agent systems, as agents are required to produce structured, role-appropriate outputs at each stage. This use of predefined procedural templates is conceptually related to compiled AI in that both approaches leverage pre-specified workflow logic to constrain and discipline LLM behavior. MetaGPT demonstrates that imposing deterministic procedural structure on LLM-based systems yields measurable improvements in output reliability and coherence.
