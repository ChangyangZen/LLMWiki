# ReAct Framework

**Type:** method

A framework synergizing reasoning and acting in language models, used in embodied agent evaluations in conjunction with SafeAgentBench.

---

ReAct is an LLM agent framework that synergizes reasoning and acting by interleaving natural language thought traces with action execution, enabling agents to dynamically plan and adapt their behavior based on environmental observations. The framework has been widely adopted for building LLM agents in interactive environments due to its simplicity and effectiveness in combining deliberative reasoning with grounded action. In ProbGuard's embodied agent evaluation, ReAct is used in conjunction with SafeAgentBench to implement the baseline LLM agent that ProbGuard monitors for unsafe household interactions. The integration of ProbGuard with ReAct-based agents demonstrates the framework's compatibility with widely used LLM agent architectures.
