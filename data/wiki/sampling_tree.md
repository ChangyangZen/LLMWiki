# Sampling Tree

**Type:** concept

A tree structure representing all possible token sequences generated by an LLM autoregressive process, with edges annotated by token probabilities across prompt, output, and interpretation stages.

---

A sampling tree is a formal tree structure used to represent all possible token sequences that can be generated by an LLM's autoregressive process. Each node in the tree corresponds to a partial token sequence, and each directed edge is annotated with the conditional probability of the next token given the preceding context. The tree spans all three stages of the generation pipeline: the prompt formulation by an actor, the LLM's token-by-token output generation, and any subsequent interpretation phase. By capturing the full probability space of token sequences, the sampling tree serves as the foundational object over which filter and objective functions operate to define uncertainty measures.
