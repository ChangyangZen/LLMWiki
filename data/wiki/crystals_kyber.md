# CRYSTALS-Kyber_Security

**Type:** dataset

An AFP project formalizing security reasoning for CRYSTALS-Kyber within the CryptHOL-based probabilistic framework, containing 64 theorems used to evaluate AutoReal-Prover.

---

CRYSTALS-Kyber_Security is an Archive of Formal Proofs project that formalizes security arguments for the CRYSTALS-Kyber post-quantum key encapsulation mechanism within the CryptHOL-based probabilistic security framework in Isabelle/HOL. It contains 64 theorems and is used in the AutoReal paper as one of three AFP evaluation projects to assess the generalization of AutoReal-Prover to security-focused formal verification tasks outside the seL4 training domain. The project's reliance on probabilistic reasoning and cryptographic abstractions makes it a challenging test case for automated theorem proving methods. Its inclusion in the evaluation suite reflects the broader goal of AutoReal to support industrial-scale verification across diverse formal security developments.
