# ShieldAgent

**Type:** method

A shielding architecture that wraps LLM agents with verifiable safety policy reasoning using Markov logic networks for probabilistic rule evaluation.

---

ShieldAgent is a shielding architecture for LLM agents that wraps agent systems with verifiable safety policy reasoning using Markov logic networks for probabilistic rule evaluation and enforcement. The framework maintains a set of safety policies expressed as probabilistic logical rules and intervenes in agent decision-making when proposed actions are predicted to violate these policies. ShieldAgent represents an approach that combines probabilistic reasoning with formal policy specification to provide safety guarantees for LLM agents. It serves as a point of comparison in the broader landscape of LLM agent safety frameworks alongside ProbGuard, AgentSpec, and GuardAgent.
