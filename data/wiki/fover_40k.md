# FoVer-40K Dataset

**Type:** dataset

A PRM training dataset of 40K steps with step-level binary error labels, created from formal logic and theorem proving tasks using Z3 and Isabelle verifiers.

---

FoVer-40K is a PRM training dataset consisting of approximately 40,000 reasoning steps annotated with binary step-level error labels, constructed using the FoVer framework. The dataset is derived from two types of formal reasoning tasks: multi-step first-order logic deduction sourced from the FLDx2 dataset and formal theorem proving tasks derived from GSM8K, with solutions generated by Llama 3.1 8B and Qwen 2.5 7B. Step-level correctness labels are assigned automatically using the Z3 SMT solver for logic tasks and the Isabelle/HOL theorem prover for theorem proving tasks, ensuring high label accuracy without human intervention. FoVer-40K is used to fine-tune LLMs into FoVer-PRMs, which demonstrate strong performance on both formal and informal reasoning benchmarks.
